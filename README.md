## Preprocessing
Data was augmented using computer vision python package to expand the dataset size , so that a better model could be trained
Following augementation operations were done
> 1. Rotation
> 2. Brightness
> 3. Vertical Flip
> 4. Horizontal Flip
> 5. Channel Shift
> 6. Horizontal Shift
> 7. Vertical Shift
